# India Crime Intelligence Dashboard

An interactive crime analytics dashboard built using Python, Dash, Plotly, and Pandas.

## Features

- Crime analysis by city
- Crime type distribution
- Gender-based analysis
- Age-group analysis
- Case closure statistics
- Interactive filters and drill-down insights

## Technologies Used

- Python
- Dash
- Plotly
- Pandas
- Dash Bootstrap Components

## Project Structure

India-Crime-Dashboard/
├── crime_dashboard.py
├── crime_dataset_india.csv
├── requirements.txt
└── README.md

## Installation

pip install -r requirements.txt

## Run

python crime_dashboard.py
